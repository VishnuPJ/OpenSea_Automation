# OpenSea_Automation
Automated bulk listing into OpenSea marketspace.

Install selenium using "pip install selenium". 

The is still under development and issues caused by network anomaly is not handled.

This code by default mints in "Polygon" blockchain. Modify the code to mint in "Ethereum".

Also modify the "properties" accordingly.

Put the images/gifs to be uploaded in "images" folder  in the same directory as the python file.
